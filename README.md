# Digital Forensics Lab - Ransomware Investigation

**Projet de forensics digital** : Analyse post-mortem complète d'une compromission ransomware.

## Investigateur

**Thomas FERET** (ThomasWEBDEV)  
Étudiant en cybersécurité - Spécialisation DFIR  
Recherche alternance : SOC Analyst / Incident Responder

## Objectif

Démontrer des compétences DFIR (Digital Forensics & Incident Response) à travers :
- Analyse mémoire (Volatility 3)
- Analyse disque (Autopsy)
- Reconstruction de timeline d'attaque
- Extraction d'IOCs
- Mapping MITRE ATT&CK
- Rapports techniques professionnels

## Outils utilisés

- Volatility 3 (memory forensics)
- Autopsy (disk forensics)  
- Python 3 (automation)
- Plaso/log2timeline (timeline analysis)

## Méthodologie

SANS PICERL :
1. Preparation - Setup lab
2. Identification - Triage initial
3. Containment - IOCs extraction
4. Eradication - Root cause analysis
5. Recovery - Remediation
6. Lessons Learned - Reporting
